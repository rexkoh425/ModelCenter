# BLIP Caption Utilities

This folder hosts helpers built around the BLIP captioning checkpoints. The current entry point `blip_caption_renamer.py` reads a YAML config, loads the BLIP model, and renames/updates caption text files (or image filenames) according to the generated descriptions. Use it to clean noisy datasets or enforce naming conventions before feeding data to other pipelines.

## Quick start

1. Install dependencies (torch, torchvision, transformers, Pillow, PyYAML) inside your environment.
2. Edit `blip_caption_renamer_config.yaml` to point at the source image folder and tweak BLIP parameters (device, beam search, prompt prefix, etc.).
3. Run from the repo root:
   ```bash
   python BLIP/blip_caption_renamer.py --config BLIP/blip_caption_renamer_config.yaml
   ```

The script logs progress and writes the updated captions or renamed files in-place. Adjust the config's dry-run/overwrite settings to preview changes safely.
