# CycleGAN Video Translator

This module contains `cyclegan_video.py`, a command-line tool that loads one of the official CycleGAN generator checkpoints (horse2zebra, summer2winter, monet2photo, etc.) and applies it frame-by-frame to an input video. It supports side-by-side comparison output, FPS overrides, frame limits, and device/precision toggles.

## Usage

1. Install dependencies:
   ```bash
   pip install torch torchvision pillow opencv-python numpy pyyaml
   ```
2. Edit `cyclegan_video_config.yaml` with your input video path, mapping name, and optional export settings.
3. Run from the repo root:
   ```bash
   python CycleGAN/cyclegan_video.py
   ```

The script writes the translated video alongside the source clip (or as an SxS composite if `side_by_side: true`).
